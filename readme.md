# tslint-jike-node

## Install

```
$ npm install --save-dev tslint-jike-node
```


## Usage

Add the following to your [TSLint config](https://palantir.github.io/tslint/usage/configuration/):

```json
{
	"extends": "tslint-jike-node"
}
```
